Программа Консольный чат.

Чат содержит три класса. Сам класс Чат, клас Пользователь и класс Сообщение.

Описание работы:
В стартовом меню на выбор даётся выбор из регистрации, входа и закрытии программы.

Регистрация состоит из создания логина, пароля и имени. Данные вводятся поочерёдно латинскими(англ), иначе вывод имени будет некоректным.
После регистрации выводится стартовое меню и можно зарегистрировать нового пользователя или войи под существующим.

При входе зарегистрированого участника открывается меню пользователя.
Меню пользователя состоит из Показа сообщений, Отправки сообщений, Показа имён зарегистрированных участников и Выхода из меню.

При показе сообщений выводятся все сообщения, в том числе собственные.

При отправке сообщений предлагается выбор из Персонального - нужно ввести имя пользователя, и Общего - нужно ввести all. 
Далее набор сообщения латинскими буквати, иначе вывод сообщения будет некоректным.
